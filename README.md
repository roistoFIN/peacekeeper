# peacekeeper
An application that helps during an active conflict. In-the-moment conflict coaching.

Project Goal: Self-help conflict coaching that guides a structured process rather than promising clinical treatment.

Unique Value: An interactive "SOS" tool that uses Generative AI (Gemini) to lower cognitive load during arguments by suggesting neutral language and identifying underlying needs.

The Tech: A modern GCP-native stack optimized for cost and performance, leveraging Vertex AI and Cloud Run.
